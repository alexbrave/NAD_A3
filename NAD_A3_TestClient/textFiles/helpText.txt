Welcome to the NAD A3 Logger Test Client.
Below you will find the commands that allow the testing of the logger.
Simply enter the command and you will be prompted for any other input, if needed,
otherwise the command will execute and you will see a status message.

manual 	  - you will be prompted to manually enter a log message to send to the client
			as well as the log level that you wish to attatch to the message
automatic - this will send an assortement of different log messages	
noise  	  - this will send many different log messages of different levels to the 
			logger to test if it is able to effectively manage a service that is 
			sending too many logs in a short span of time
quit	  - quit the program
